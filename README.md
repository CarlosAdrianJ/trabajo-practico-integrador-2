# Food Store Console

Sistema de gestión de pedidos de comida desarrollado como Trabajo Práctico Integrador de Programación 2.

El proyecto está construido como una aplicación de consola en Java 21, utilizando Programación Orientada a Objetos, JDBC puro, patrón DAO, servicios y base de datos MySQL.

## Tecnologías utilizadas

- Java 21
- Gradle
- MySQL
- MySQL Workbench
- JDBC puro
- Programación Orientada a Objetos
- Patrón DAO
- Aplicación de consola
